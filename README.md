# Superstore Data Normalization & Dashboard Analysis

## Project Overview
This project demonstrates how raw business data can be transformed into a structured relational model through **data normalization**, and then analyzed through an interactive **Excel dashboard**.

The goal is to convert a flat dataset into a **relational data model**, improve data structure, and extract meaningful insights using Excel tools.

---

## Objectives
- Normalize the Superstore dataset into structured tables
- Build a relational data model
- Perform data cleaning and transformation using Power Query
- Create calculated columns and measures
- Build an interactive dashboard for analysis

---

## Tools Used

- Microsoft Excel
- Power Query
- Power Pivot
- Pivot Tables

---

## Data Normalization

The original dataset was transformed into multiple related tables including:

- Orders
- Customers
- Products
- People
- Returns

This process reduces redundancy and improves data integrity.

---

## Dashboard Insights

The dashboard provides insights into:

- Total Sales
- Profit Analysis
- Sales by Region
- Category Performance
- Customer Segments
- Sales Trends
